HTML Tag Explanation
<meta> Specifies the viewport settings, ensuring the webpage is responsive.
<div class="container"> Wraps content in a centered container with Bootstrap's default padding and margins.
<h1 class="text-primary"> Creates a heading with Bootstrap's primary text color.
<div class="row"> Defines a row in the grid system to structure content in columns.
<div class="col-md-6"> Defines a column that spans 6 out of 12 columns on medium and larger screens.
<table> Creates a table to display data in rows and columns.
<thead class="table-dark"> Defines the table header with a dark background and light text.
